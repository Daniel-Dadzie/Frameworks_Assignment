COVID-19 Data Explorer
This project is part of the Python Frameworks Assignment. It uses a sample COVID-19 dataset from GitHub to demonstrate basic data analysis and visualization using pandas, matplotlib, seaborn, and Streamlit.

Features
Load and clean real-world COVID-19 data
Visualize global case trends over time
Compare top countries by total cases
Interactive dashboard with year filtering
Tools Used
Python 3.7+
pandas
matplotlib
seaborn
Streamlit
How to Run
Clone this repo
Install dependencies:
pip install pandas matplotlib seaborn streamlit

Run the app:
streamlit run app.py
